# 烟台文化旅游职业学院学生信息核验管理系统

本项目是面向“烟台文化旅游职业学院”的学生信息核验管理系统，用于导入标准学生库和待核对学生表，按学号自动比对学生基础信息，并输出异常结果。

## 学校品牌信息

- 学校中文名称：烟台文化旅游职业学院
- 学校英文名称：YANTAI VOCATIONAL COLLEGE OF CULTURE AND TOURISM
- 系统名称：学生信息管理系统
- 副标题：校园学生基础信息数字化核验平台
- 校徽图片路径：`assets/school-logo.jpg`

## Excel 导入模板字段

系统支持 `.xlsx` 和 `.xls` 文件，自动读取第一个工作表。

必填表头：

- 姓名
- 学号
- 性别
- 班级
- 联系方式

可选表头：

- 年龄
- 家庭住址
- 备注

导入时会自动去除字段前后空格，学号和联系方式会按字符串处理，联系方式会去除空格。

## 核验规则

点击“开始比对”后，系统按学号比对标准学生库与待核对表：

- 学号为空：无效数据
- 学号重复：重复学号
- 待核对表中的学号在标准库中不存在：未录入学生
- 标准库中存在学号，但姓名、性别、年龄、班级、联系方式、家庭住址任一字段不同：信息不一致
- 标准库中存在但待核对表缺少对应学号：待核对表缺失
- 所有核心字段一致：通过

备注字段不参与自动比对。

## 功能

- 登录后进入学生信息核验后台
- 导入标准学生库 Excel
- 导入待核对表 Excel
- 自动核验与异常结果展示
- 查看标准学生库、待核对表、核对结果
- 手动新增、编辑、删除学生基础信息
- 搜索、班级筛选、排序
- 统计学生总数、核对通过、异常数据、重复学号
- 导出核对结果，优先导出 Excel，解析库不可用时导出 CSV
- 数据保存到浏览器 `localStorage`

## 运行方式

直接双击打开 `index.html`，或在浏览器中打开该文件即可使用。

默认演示登录账号：

- 账号：`admin`
- 密码：`YTCT@2026`

本项目通过 CDN 引入 SheetJS：

```html
https://cdn.jsdelivr.net/npm/xlsx/dist/xlsx.full.min.js
```

如果无法联网，Excel 导入和 Excel 导出会受影响；核对结果导出会降级为 CSV。

## 数据保存方式

数据保存在当前浏览器的 `localStorage` 中，包括：

- `standardStudents` 标准学生库
- `checkStudents` 待核对学生表
- `verificationResults` 核对结果

清理浏览器缓存或更换浏览器可能导致本地数据丢失。

## 注意事项

- 登录功能为前端本地演示登录，用于项目展示和页面访问控制示意，不等同于服务端安全认证。
- 性别建议填写“男”或“女”，其他内容会在核验中标记为格式异常。
- 年龄可为空；如果填写，必须是合理数字。
- 联系方式建议填写 11 位中国大陆手机号。
- 手动新增的数据默认进入标准学生库。
- 重置数据会覆盖当前本地标准库、待核对表和核对结果。
